# vpsh.sh - VPS 管理工具集合脚本

## 功能描述
功能丰富的 VPS 服务器管理工具集合，提供现代化的交互界面。

## 主要特点
- 🎨 现代化彩色界面设计
- 📱 自适应终端宽度
- 🖥️ 美观的边框和分隔符
- 🎯 智能颜色检测和兼容性
- 📋 丰富的系统管理功能

## 界面特色
- 支持多种颜色主题
- 动态调整显示宽度
- 隐藏光标优化体验
- 现代化 Unicode 边框

## 使用方法
```bash
bash <(curl -Ls s.bbql.de)
```

## 功能模块
脚本包含多个实用的 VPS 管理功能（具体功能需查看完整脚本内容）

## 适用系统
- 支持彩色终端的 Linux 系统
- 兼容各种终端环境
- 自动适配终端能力

## 特点
- 用户友好的交互界面
- 智能终端检测
- 现代化视觉设计
- 功能模块化设计
